# Row Group Codes (p1217 / p1218 / p1219 / p1186 / p1759)

Values usable for **row grouping** on the Y-axis. Assign in order to the up-to-5 row-level parameters: **`p1217`** (Level 1) → **`p1218`** (Level 2) → **`p1219`** (Level 3) → **`p1186`** (Level 4) → **`p1759`** (Level 5).
`p1217` (Row Level 1) is mandatory; it defaults to `RowGrpRiskType`. The remaining levels are optional. Use the Code exactly as written.
